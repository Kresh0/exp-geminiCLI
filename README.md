# Gemini CLI Showcase

An interactive, high-impact landing page built to showcase the autonomous coding capabilities and multi-agent orchestration of **Gemini CLI**.

![Gemini CLI Showcase Preview](https://github.com/user-attachments/assets/placeholder-image.png)

## 🚀 Features

- **Interactive Terminal Hero:** Realistic simulation of Gemini CLI's autonomous workflow (Research → Strategy → Execution → Validation).
- **Draggable Code Diff:** A side-by-side comparison tool demonstrating legacy code refactoring into modern, type-safe TypeScript and Tailwind CSS v4.
- **Multi-Agent Orchestration Visualizer:** A dynamic SVG-based animation showing the delegation of tasks between specialized sub-agents.
- **Modern Tech Stack:** Built with the latest stable versions of Next.js, React, and Tailwind CSS.
- **High-Performance Animations:** Fluid transitions and scroll-triggered effects powered by Framer Motion.
- **One-Click Installation Copy:** Integrated clipboard functionality for the Gemini CLI installation command.

## 🛠️ Tech Stack

- **Framework:** [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🏁 Getting Started

### Prerequisites

- Node.js 22.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/gemini-cli-showcase.git
   cd gemini-cli-showcase
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Build & Deployment

To create an optimized production build:

```bash
npm run build
```

This project is optimized for deployment on [Vercel](https://vercel.com/new?utm_source=create-next-app&utm_medium=appdir-template&utm_campaign=create-next-app).
